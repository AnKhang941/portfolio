React Portfolio Website

Responsive portfolio website with dark mode and smooth animations.

Project Overview

This React-based portfolio website showcases modern web development practices with a focus on user experience. It features a responsive design that works seamlessly across all devices, a dark mode toggle with local storage persistence, and smooth CSS animations for enhanced interactivity.

Technologies Used

React CSS3 JavaScript (ES6+) React Router Framer Motion

Key Features

  • Responsive design for all screen sizes
  • Dark/Light mode toggle with persistence
  • Smooth page transitions and animations
  • Single Page Application (SPA) routing
  • Optimized performance with lazy loading
  • SEO-friendly with React Helmet
