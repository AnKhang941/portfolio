Flutter E-commerce App

A cross-platform e-commerce application built with Flutter.

Project Overview

This Flutter-based e-commerce app provides a seamless shopping experience across iOS and Android platforms. It features real-time inventory management, secure user authentication via Firebase, and an intuitive UI designed for optimal user engagement.

Technologies Used

Flutter Firebase Dart Provider (State Management)

Key Features

  • User authentication and profile management
  • Product catalog with search and filtering
  • Shopping cart and checkout process
  • Order history and tracking
  • Real-time inventory updates
