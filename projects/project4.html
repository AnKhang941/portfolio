.NET Web Application

Web application for project management with user roles and task tracking.

Project Overview

This ASP.NET Core web application provides a comprehensive project management solution. It includes user authentication, role-based permissions, task assignment and tracking, project timelines, and reporting features. The application uses Entity Framework for data access and SQL Server for data storage.

Technologies Used

ASP.NET Core C# Entity Framework SQL Server Bootstrap

Key Features

  • User authentication and role management
  • Project creation and management
  • Task assignment and progress tracking
  • Team collaboration tools
  • Dashboard with project analytics
  • Export reports to PDF/Excel
